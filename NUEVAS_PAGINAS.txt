✅ ANTES (Versión 1.0):
   └─ Todo en 1 página: index.html con secciones (#inicio, #ruleta, #comunidad)

✅ AHORA (Versión 2.0):
   ├─ index.html → Página de INICIO
   ├─ ruleta.html → Página de RULETA
   └─ comunidad.html → Página de COMUNIDAD

✅ BENEFICIOS:
   • Mejor organización y estructura
   • Navegación más clara
   • URLs distintas para cada sección
   • Mantenimiento más fácil
   • Escalable para agregar más páginas
   • Mejor para SEO (si se despliega online)

┌────────────────────────────────────────────────────────────────────────────┐
│                        CÓMO FUNCIONA AHORA                                │
└────────────────────────────────────────────────────────────────────────────┘

1. NAVEGACIÓN ENTRE PÁGINAS
   ─────────────────────────
   • Header con 3 links: Inicio, Ruleta, Comunidad
   • El link activo se resalta automáticamente
   • Cada página tiene botones internos para navegar

2. COMPARTIR EL MISMO JavaScript
   ────────────────────────────────
   • js/main.js se carga en las 3 páginas
   • Funciones verifican si elementos existen:

     if (document.getElementById('wheelCanvas')) {
         // Solo ejecutar en ruleta.html
     }

3. COMPARTIR EL MISMO CSS
   ──────────────────────
   • css/main.css contiene TODOS los estilos
   • Se usa en index.html, ruleta.html y comunidad.html
   • Las clases y variables son globales

4. COOKIES Y COMENTARIOS
   ─────────────────────
   • Se guardan en localStorage (navegador)
   • Se sincronizar entre páginas automáticamente
   • Pueden ver/escribir desde cualquier página

5. HISTORIAL DE RETOS
   ──────────────────
   • Se guarda en localStorage
   • Accesible desde ruleta.html
   • Persiste entre página e navegaciones

FLUJO 1: Usuario nuevo

┌────────────────────────────────────────────────────────────────────────────┐
│                          SOPORTE Y TROUBLESHOOTING                         │
└────────────────────────────────────────────────────────────────────────────┘

❓ ¿Los links no funcionan?
→ Asegúrate de estar usando un servidor local, no abriendo archivos directamente

❓ ¿Los comentarios no se guardan entre páginas?
→ Verificar que localStorage esté habilitado en el navegador

❓ ¿La ruleta no aparece?
→ Abre ruleta.html directamente, no intentes verla desde index.html

❓ ¿Cómo eliminar comentarios?
→ Abre DevTools (F12) → Application → LocalStorage → Busca "comentarios"

❓ ¿Cómo agregar más retos?
→ Abre js/main.js, busca "const retos = [" y agrega más elementos
